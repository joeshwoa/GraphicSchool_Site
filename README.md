# GraphicSchool Site
GraphicSchool Site is a web-based platform designed for a graphic design learning community. The website offers tutorials, courses, and resources for aspiring and professional graphic designers. Built with modern web development technologies, the site provides a clean, user-friendly interface to help users navigate through a variety of design-related content.

## Overview
The GraphicSchool Site project aims to create a comprehensive platform for graphic design education. It provides users with access to various resources including video tutorials, articles, design tools, and more. Whether you're a beginner or an advanced designer, the site has a range of content tailored to different skill levels.

## Features
Tutorials and Courses: Access step-by-step guides, video lessons, and complete courses covering design principles, software usage, and industry best practices.

Responsive Design: Optimized for viewing on desktop and mobile devices.

User Accounts: Allows users to sign up, log in, and track their course progress.

Interactive Learning: Includes quizzes and exercises to reinforce learning.

Resource Library: Offers downloadable design templates, mockups, and tools.

## Installation
To run the GraphicSchool Site on your local machine:

1. Clone the repository: ```git clone https://github.com/joeshwoa/GraphicSchool_Site.git```
2. Navigate into the project directory: ```cd GraphicSchool_Site```
3. Install necessary dependencies: ```flutter pub get```
4. Start the local development server: ```flutter run```
5. Open the site in your browser: ```http://localhost:[port]```

## Usage
Register an Account: Sign up on the platform to start learning and tracking your progress.

Browse Courses: Select from a variety of graphic design courses, from beginner to advanced levels.

Access Resources: Download design assets like templates, icons, and mockups from the resource library.

Interactive Quizzes: Take quizzes after each lesson to reinforce your understanding of the material.
